Tạo link cho khách

Vào phần Tạo link để nhập API Key, Folder ID của Google Drive, đặt mật khẩu và giới hạn số ảnh được chọn. Hệ thống sẽ tạo URL riêng gửi cho khách.

Lưu ý: vì chạy tĩnh trên Pages, mật khẩu chỉ kiểm tra ở trình duyệt (không chống được người rành kỹ thuật). Nếu bạn cần bảo vệ thật sự, nên dùng server/ serverless.

Khách truy cập

Nếu bạn là khách, bạn sẽ nhận 1 đường link dạng #token=.... Mở link, nhập mật khẩu là thấy ảnh.

Ví dụ: https://tenban.github.io/gallery.html#token=...

Trình tạo link cho khách

Hướng dẫn lấy API Key & Folder ID
  1. Mở Google Cloud Console → tạo Project → bật Google Drive API → tạo API Key.
  2. Trên Google Drive, để folder chứa ảnh ở chế độ Anyone with the link – Viewer.
  3. Lấy Folder ID từ URL thư mục: dạng /folders/<ID>.

Cảnh báo bảo mật: Trang này là static (không có server). Password chỉ được kiểm tra trong trình duyệt; người am hiểu có thể đọc token. Dùng như một lớp “khóa nhẹ”.
